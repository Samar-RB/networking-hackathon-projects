# Networking Hackathon Projects

## Overview
This repository contains two client-server networking projects implemented in Python as part of networking hackathon assignments.

The projects focus on protocol design, socket programming, TCP/UDP communication, multithreading, and interoperability between clients and servers.

## Projects

### 1. Network Speed Test
A client-server application for measuring network performance using TCP and UDP transfers, including transfer speed and packet loss statistics.

### 2. Network Blackjack
A client-server Blackjack game using UDP broadcast for server discovery and TCP for game communication, including multi-round gameplay and client statistics.
